# Konnie's Tech Space

使用 [Astro](https://astro.build) 和 [Fuwari](https://github.com/saicaca/fuwari) 主題建立的個人技術部落格。

## 指令

| 指令                       | 說明                                     |
|:---------------------------|:-----------------------------------------|
| `pnpm install`             | 安裝依賴                                 |
| `pnpm dev`                 | 啟動本地開發伺服器 `localhost:4321`       |
| `pnpm build`               | 建置正式版網站至 `./dist/`               |
| `pnpm preview`             | 本地預覽建置結果                          |
| `pnpm new-post <filename>` | 建立新文章                                |
